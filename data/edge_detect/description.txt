Create a module named top_module that detects the rising edge of a 1-bit input signal. The module has a clock input 'clk', an active-high synchronous reset 'reset', a 1-bit input 'in', and a 1-bit output 'rise'. On each rising clock edge, if reset is high, the output should be 0. Otherwise, the output 'rise' should be 1 for exactly one clock cycle whenever 'in' transitions from 0 to 1 (i.e., 'in' was 0 on the previous clock cycle and is 1 on the current clock cycle). At all other times 'rise' should be 0.
