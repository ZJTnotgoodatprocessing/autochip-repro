Create a module named top_module that implements an 8-bit priority encoder. The module has an 8-bit input 'in' and two outputs: a 3-bit output 'pos' indicating the bit position of the highest-priority (most significant) bit that is set to 1, and a 1-bit output 'valid' which is 1 when at least one input bit is set. Priority goes from MSB (bit 7, highest priority) to LSB (bit 0, lowest priority). When no input bits are set (in == 8'b0), 'valid' should be 0 and 'pos' should be 3'b000. For example: in=8'b10000000 -> pos=3'd7, valid=1; in=8'b00001010 -> pos=3'd3, valid=1; in=8'b00000001 -> pos=3'd0, valid=1; in=8'b00000000 -> pos=3'd0, valid=0.
