Create a module named top_module that implements 8-bit signed addition with overflow detection. The module has two 8-bit signed inputs 'a' and 'b', an 8-bit output 'sum' which is the sum of a and b, and a 1-bit output 'overflow'. The overflow flag should be 1 when signed overflow occurs: this happens when adding two positive numbers produces a negative result, or when adding two negative numbers produces a positive result. In terms of implementation, overflow can be detected by checking if the carry into the MSB differs from the carry out of the MSB, which is equivalent to: overflow = (a[7] == b[7]) && (sum[7] != a[7]). Note: treat all values as 2's complement signed numbers.
