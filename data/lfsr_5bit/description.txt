Create a module named top_module that implements a 5-bit linear feedback shift register (LFSR). The module has a clock input 'clk', an active-high synchronous reset 'reset', and a 5-bit output 'q'. On reset, q should be initialized to 5'b00001. On each clock cycle (when not in reset), the register shifts left by one bit. The new bit shifted into position 0 is the XOR of bit 4 and bit 2 of the current state (i.e., new_bit = q[4] ^ q[2]). So the next state is: q_next = {q[3:0], q[4] ^ q[2]}. The first few states after reset should be: 00001, 00010, 00100, 01001, 10010, 00101, ...
