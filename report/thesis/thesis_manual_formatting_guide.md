# 论文人工排版指导包

> 创建日期：2026-05-05
> 用途：指导用户将 Markdown 初稿内容手工整理进北航本科论文 Word 模板

---

## 1. Word v1 的定位

`report/thesis/thesis_draft_v1.docx` 是**内容整合版**，不是格式终稿。

- ✅ 可用于检查内容完整性（六章 + 摘要 + 致谢 + 参考文献 + 附录）
- ✅ 可用于检查图表齐全性（16 张图已嵌入）
- ✅ 可用于检查章节顺序和逻辑衔接
- ❌ 不建议直接作为最终提交版
- ❌ 封面、页眉页码、字体行距、图表编号、三线表、参考文献格式均需人工调整

后续工作流程：
1. 打开 `北航本科生论文模板.docx`
2. 删除模板中的示例论文内容
3. 参照本指导包，将各章 Markdown 内容逐章粘入 Word
4. 按下方检查清单逐项调整格式

- [ ] 中文摘要标题格式正确（"摘 要"）

- [ ] 英文摘要标题格式正确（"Abstract"）
- [ ] Key Words 格式正确

### 3.4 目录

- [ ] 使用 Word 内建"引用 > 目录"自动生成
- [ ] 确认所有章标题和节标题使用了正确的 Heading 样式
- [ ] 目录页码正确

### 3.5 正文格式

- [ ] 章标题使用 Heading 1（例："第 1 章 绪论"）
- [ ] 节标题使用 Heading 2/3/4
- [ ] 正文字体：中文宋体，英文 Times New Roman
- [ ] 正文字号：小四（12pt）
- [ ] 行距：1.5 倍或按模板要求
- [ ] 段落首行缩进 2 字符

### 3.6 图表

- [ ] 图题在图下方，居中，格式"图 X.Y 标题"
- [ ] 表题在表上方，居中，格式"表 X.Y 标题"
- [ ] 表格使用三线表（仅顶线、栏目线、底线）
- [ ] 所有图编号与正文引用一致
- [ ] 不含 multi-turn v1 废弃图
- [ ] 图片不跨页或图题不与图片分离
- [ ] 参考 `figure_table_placement_plan.md` 中的建议

### 3.7 参考文献

- [ ] 使用 `references_verified.md` 中已核验的文献
- [ ] 格式统一为 GB/T 7714 或学校要求
- [ ] 正文中使用上角标 [1] 格式引用
- [ ] 参考 `citation_insertion_plan.md` 中的引用位置建议
- [ ] 确认无 TODO:verify 文献混入最终版
- [ ] 编号连续，不跳号

### 3.8 页眉页码

- [ ] 前置部分（摘要、目录）：罗马数字页码
- [ ] 正文部分：阿拉伯数字页码
- [ ] 页眉内容按模板要求（通常为论文题目或章标题）

### 3.9 附录

- [ ] 附录编号：附录 A、B、C、D
- [ ] 实验命令中无 API key / base_url / token
- [ ] multi-turn v1 标记为 Superseded
- [ ] 代码模块仅列索引，不粘贴大段源码

### 3.10 最终检查

- [ ] 已删除模板中原始示例论文内容
- [ ] 已删除所有 TODO 标记或转为明确待办
- [ ] 全文术语一致（zero-shot, retry-only, feedback 等）
- [ ] 无虚构内容
- [ ] 无 superseded 数据引用
