## 目录说明

本目录用于存放论文撰写相关材料。

- **论文模板** (`北航本科生论文模板.docx`) 位于项目根目录，不复制到本目录，避免二进制文件膨胀仓库
- **写作计划** 位于 `notes/thesis_writing_plan.md`
- **权威实验索引** 位于 `notes/authoritative_experiment_index.md`
- **图表来源** 位于 `outputs/reports/` 和 `report/figures/`

## 注意事项

1. 论文中所有实验数据引用必须经 `notes/authoritative_experiment_index.md` 确认
2. Multi-turn v1 (`9760eb4`) 的结论 **不可引用**，已被 v2 替代
3. 图表应优先使用 `outputs/reports/` 中带 `_v2` 后缀的版本（如 `fig_multiturn_comparison_v2.png`）
