## 4. 全文润色

### 润色原则

- 将机械化的"首先…其次…最后…"转为自然衔接
- 合并过短段落
- 去除绝对化表述
- 保持正式学术语气
- **未修改任何实验数据**

### 重点润色章节

| 章节 | 润色内容 |
|------|---------|
| 摘要 | 去除过度工整的四段式结构感，缩短冗余描述 |
| 第1章 | §1.3 研究内容改为两层面自然段，§1.4 贡献改为三层面自然段，§1.5 组织结构压缩为一段 |
| 第4章 | 全部三级标题改为加粗段首标题，语言自然化 |
| 第5章 | 消除"总体结果/关键发现/小结"三段式，改为连贯分析 |
| 第6章 | §6.4 五个子节合并为一段自然段 |
| 结论页 | 保持不变（已经较精炼） |
